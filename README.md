# zemoga-challange-java
Portfolio Java Web App

## About the project
This service provides information about users and their last tweets.

### Technologies
- JDK 11
- Maven 3+
- Spring Boot version: 2.6.5
- Front: React
- Build tool: Gradle 7.4.1


### build
- gradle build

### run
- gradle run

### Running on url
<p>http://localhost:8080/</p>
